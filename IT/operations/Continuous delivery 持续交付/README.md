# 持续交付

- **版本控制系统** GitLab GiHub
- **持续集成工具** Jenkins 把代码从平台抓取到本地
- **自动化部署交付工具** Ansible Saltstack Chef 将代码从本体，推送到服务器交付



**持续交付 = 版本控制系统 + 持续集成工具 + 部署工具**



####课程安排

- Gitlab代码仓库搭建和使用流程
- Ansible环境配置与Playbook编写规范
- Jenkins环境构建与任务编写
- Freestyle任务实现静态网站部署交付
- Pipeline任务实现WordPress部署交付
