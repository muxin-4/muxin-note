#### 向量

- 为什么线性代数这么重要？从研究一个数拓展到研究一组数  一个数：666   一组数：（6， 66， 666）

- 一组数的基本表示方法--向量（Vector）
- 向量（Vector）是线性代数研究的基本元素

- 一组数有什么用？   最早的时候为了表示方向
  - 为了研究方便，我们定义向量都是从原点开始
  - 但是，顺序是重要的
  - （4，3）和（3， 4）截然不同

- 如果只是表示方向，最多三个维度就够了
- 更加抽象的：n 维向量
