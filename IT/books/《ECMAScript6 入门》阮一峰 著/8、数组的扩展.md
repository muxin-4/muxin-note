# 8、数组的扩展

### 1.  扩展运算符

```
function push(array, ...items) {
  array.push(...items);
}

function add(x, y) {
  console.log('x', x);
  console.log('y', y);
  return x + y;
}

const numbers = [4, 38];
add(...numbers) // 42
```

#### 1.2 扩展运算符实际应用

- 数组合并

```javascript
arr1 = [1,2,3]
arr2 = [4,5,6]
arr3 = [7,8,9]
 
arr4 = [...arr1, ...arr2, ...arr3] //[1,2,3,4,5,6,7,8,9]  用于数组合并
```

- 数组解耦

  ```JavaScript
  let [a,b, ...c] = list
  a //a
  b //b
  c //[c,d,e]
  
  ```
