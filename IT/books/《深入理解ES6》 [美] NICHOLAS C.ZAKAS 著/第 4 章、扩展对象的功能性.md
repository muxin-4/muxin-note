# 第 4 章、扩展对象的功能性

#### 1.简写： 属性初始值

例：创建一个对象

ES5

```javascript
function createPerson(name, age) {
    return {
        name: name,
        age: age
    }
}
```

ES6：当对象字面量只有一个属性的名称时，Js引擎会在可访问作用域中查找其同名变量

```javascript
function createPerson(name, age) {
    return {
        name,
        age
    }
}
```

ES6的好处：有助于消除命名错误

#### 2.简写：对象方法

ES5

```javascript
var person = {
    name: "Nicholas",
    sayName: function() {
        console.log(this.name);
    }
}
```

ES6: 简写方法可以使用super关键字

```javascript
var person = {
    name: "Nicholas",
    sayName() {
        console.log(this.name);
    }
}
```
