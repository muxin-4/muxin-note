#### 暴力破解

- Google黑语法
  - inurl：搜索url网址中包含的指定字符串
    - 百度搜索`inurl:login.php`
  - intitle：搜索网页中的标题名是否包含指定字
    - 百度搜索`intitle:登陆`
  - intext：搜索网页正文内容中的指定字符
    - 百度搜索`intext:登录`
