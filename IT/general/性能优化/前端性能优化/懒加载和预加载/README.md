## 懒加载（延迟加载）
1.懒加载原理
- 图片进入可视区域之后请求图片资源
- 对于电商等图片很多，页面很长的业务场景适用
- 减少无效资源的加载
- 并发加载过多会阻塞js的加载，影响网站的正常使用

2.实现
监听scroll事件，在scroll事件的回调中，去判断我们的懒加载的图片是否进入可视区域
