- 有一定 Java 编程功底、了解并使用过 Spring 框架

- 工作中需要使用，想系统学习和全面掌握 RabbitMQ 的小伙伴
- 一起探讨交流 MQ 中间件如何应用到系统架构设计之中的小伙伴

### 学习方法

- 从基础开始，一步一步跟着老师的节奏、仔细练习
