# 第 5 章 CSS效果

# 1. 效果属性

1. [box-shadow](https://developer.mozilla.org/zh-CN/docs/Web/CSS/box-shadow)
- 营造层次感（立体感）
- 充当没有宽度的边框
- 特殊效果（例如机器猫）
2. text-shadow
- 营造层次感（立体感）
- 印刷品质感
3. border-radius
- 圆角矩形
- 圆形
- 半圆/扇形
- 一些奇怪的角角
5. background
- 纹理、图案
- 渐变
- 雪碧图动画
- 背景图尺寸适应

6. clip-path
- 对容器进行裁剪
- 常见几何图形
- 自定义路径
