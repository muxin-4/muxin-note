### 2. 过渡动画和关键帧动画的区别

过渡动画需要有状态变化（比如鼠标移入）

关键帧动画不需要状态变化

关键帧动画控制更精细

### 3. 如何实现逐帧动画

使用关键帧动画

去掉补间（steps）
