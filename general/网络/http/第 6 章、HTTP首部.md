# 第 6 章、HTTP首部

### 一、HTTP/1.1 首部字段一览

####1、 通用首部字段

| 首部字段名        |                       说明 |
| ----------------- | -------------------------: |
| Cache-Control     |             控制缓存的行为 |
| Connnetion        |       逐跳首部、连接的管理 |
| Date              |             创建报文的时间 |
| Pragma            |                   报文指令 |
| Trailer           |         报文末端的首部一览 |
| Transfer-Encoding | 指定报文主体的传输编码方式 |
| Upgrade           |             升级为其他协议 |
| Via               |       代理服务器的相关信息 |
| Warning           |                   错误通知 |

####2、请求首部字段	

| 首部字段名        |                     说明 |
| ----------------- | -----------------------: |
| Accept            | 用户代理可处理的媒体类型 |
| Accept-Charset    |             优先的字符集 |
| Accept-Encoding   |           优先的内容编码 |
| Accept-Language   |   优先的语言（自然语言） |
| Authorization     |              Web认证信息 |
| Expect            |     期待服务器的特定行为 |
| From              |       用户的电子邮箱地址 |
| Host              |       请求资源所在服务器 |
| If-Match          |     比较实体标记（ETag） |
| If-modified-Since |       比较资源的更新时间 |
| If-None-          |                          |
