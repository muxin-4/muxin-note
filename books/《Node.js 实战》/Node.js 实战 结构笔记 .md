####1.这是什么样的一本书？

计算机教材类, 面向实战的Node.js入门教材

####2.整本书在谈论什么？

 本书的主要目的是教你学会如何创建和部署Node 程序，重点是Web 程序 。 Web 程序框架Express 和中间件框架Connect ，主要是因为它们的用途和社区的支持。你还会学到如何创建自动化测试，以及如何部署你的程序。
